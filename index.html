How to Use ReddiRecs: Product Finder

Setting Up Your ChatGPT API Key

  1. Sign up for an OpenAI account and obtain an API key from OpenAI's website.
  2. Click on the ReddiRecs extension icon in your Chrome toolbar.
  3. In the popup, paste your ChatGPT API key into the "Enter ChatGPT API key" field.
  4. Click "Save API Key". You should see a success message.

Searching for Product Recommendations

  1. Click on the ReddiRecs extension icon in your Chrome toolbar.
  2. Enter the name of the product you're interested in (e.g., "running shoes", "coffee maker").
  3. Click the "Search" button or press Enter.
  4. Wait for the results to load. This may take a few seconds.

Understanding the Results

  • TLDR Summary: At the top of the results, you'll see a brief summary of the top recommendations or opinions.
  • Product Posts: Below the summary, you'll see individual posts from Reddit discussions.
  • BIFL Tag: Posts from the "Buy It For Life" subreddit are marked with a special tag.
  • View on Reddit: Each post has a link to view the full discussion on Reddit.

Giving Feedback

We value your input! If you have any suggestions, issues, or feedback:

  1. Click on the "Give Feedback" link in the extension popup.
  2. Fill out the feedback form with your comments or concerns.

Troubleshooting

  • If you see an error message, make sure your ChatGPT API key is entered correctly.
  • Ensure you have a stable internet connection.
  • If problems persist, try reloading the extension or your browser.

Happy product hunting!
